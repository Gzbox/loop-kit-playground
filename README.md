# 🧪 Loop Kit Playground

A test project for validating [Loop Kit](https://github.com/Gzbox/loop-kit) — structured issue processing for AI coding agents.

## What is this?

This is a simple Node.js utility library used to test Loop Kit's `/loop` workflow end-to-end.

## Setup

```bash
npm install
npm test
```
