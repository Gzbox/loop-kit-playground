Utility Functions

Lightweight, dependency-free utilities for everyday JavaScript.

Aa

capitalize

Capitalize the first letter of any string.

capitalize("hello")
// → "Hello"
Σ

sum

Sum an array of numbers with safety guards.

sum([1, 2, 3, 4])
// → 10
⇔

clamp

Clamp a number within a min–max range.

clamp(15, 0, 10)
// → 10
✂️

truncate

Truncate long strings with an ellipsis.

truncate("Hello World", 8)
// → "Hello..."
🔗

slugify

Convert any string into a URL-friendly slug.

slugify("Hello World!")
// → "hello-world"
